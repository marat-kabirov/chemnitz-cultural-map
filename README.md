# 🗺️ Chemnitz Cultural Map

A full-stack web application displaying cultural locations in Chemnitz, Germany.

## ✨ Features
- Interactive map with cultural locations
- User authentication
- GeoJSON rendering

## 🛠️ Tech Stack
- **Frontend:** React, JavaScript
- **Backend:** Node.js, Express
- **Database:** MongoDB
- **Maps:** GeoJSON
